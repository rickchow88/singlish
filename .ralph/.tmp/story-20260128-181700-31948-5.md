### STORY-005: Fix Main/Boss Keyword Mapping
Status: in_progress
Depends on: None

Description:
Ensure 'kampung main' and 'action boss' parse correctly to Go's main package and function.

Goal:
'kampung main' -> package main
'action boss' -> func main()

Acceptance Criteria:
- [ ] Source file with 'kampung main' transpiles to 'package main'.
- [ ] Source file with 'action boss' transpiles to 'func main()'.
- [ ] Compiled binary runs correctly.
