### STORY-003: Support Multiple Variable Declarations
Status: in_progress
Depends on: None

Description:
Update AST and Parser to support multiple variable declarations in LetStatement.

Issue:
ast.LetStatement only supports single identifier. Fails on 'got f, err = ...'.

Technical Spec:
- Update LetStatement to have Names []*Identifier (instead of single Name).
- Update parseLetStatement to handle comma-separated identifiers.
- Update visitLetStatement in codegen to generate comma-separated variables.

Acceptance Criteria:
- [ ] Parser accepts 'got x, y = ...' syntax.
- [ ] Codegen produces valid Go multi-variable assignment.
- [ ] Existing single-variable declarations still work.
